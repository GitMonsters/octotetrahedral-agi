📋 ALL COMMAND OPTIONS
═════════════════════════════════════════════════════════════════════════════════

python3 ngvt_inspect_ai_integration.py [OPTIONS]

Options:
  --limit N              Evaluate first N questions (default: all 450)
  --level {1,2,3}       Specific difficulty level (default: all)
  --split {validation,test}  validation=has answers, test=for leaderboard
  --output FILE         Save results to JSON file
  --quick               Quick test mode (10 questions)
  --full                Full evaluation (450 questions)
  --with-docker         Use Docker for web_search and bash tools
  --no-embeddings       Disable semantic embeddings (faster)

Examples:
  python3 ngvt_inspect_ai_integration.py --quick              # 10 questions
  python3 ngvt_inspect_ai_integration.py --limit 50           # 50 questions
  python3 ngvt_inspect_ai_integration.py --limit 50 --level 1 # 50 Level 1
  python3 ngvt_inspect_ai_integration.py --full               # 450 questions
  python3 ngvt_inspect_ai_integration.py --full --output results.json

🔧 CONFIGURATION

  export NGVT_MAX_ATTEMPTS=3                     # Max reasoning attempts

Code Configuration (edit ngvt_inspect_ai_integration.py):
  semantic_match_threshold = 0.75  # 0.0-1.0 (higher = stricter)
  max_attempts = 3                 # 1-10 (more = slower but more accurate)
  use_semantic_matching = True     # True/False (True for better accuracy)

🐛 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

Problem: "Module not found" errors
Solution: pip3 install sentence-transformers inspect-evals

Problem: "HF_TOKEN not set" or "HTTP 401" when fetching data
Solution: 
  1. Go to https://huggingface.co/datasets/gaia-benchmark/GAIA
  2. Request dataset access
  3. Wait for approval (~1 day)
  4. Create token: https://huggingface.co/settings/tokens
  5. Run: export HF_TOKEN='your_token_here'

Problem: "Docker not found" when using --with-docker
Solution: 
  1. Download Docker Desktop: https://www.docker.com/products/docker-desktop
  2. Install and launch
  3. Verify: docker run hello-world
  4. Re-run with --with-docker flag

Problem: Out of memory
Solution: 
  1. Use --limit flag to process fewer questions
  2. Evaluate by difficulty level separately
  3. Close other applications

Problem: Very slow evaluation
Solution:
  1. Disable semantic embeddings: --no-embeddings
  2. Reduce semantic_match_threshold (faster but less accurate)
  3. Use Level 1 only for quicker testing


📈 INTERPRETING RESULTS
═════════════════════════════════════════════════════════════════════════════════

JSON Report Structure:
{
  "timestamp": "2026-02-15T18:30:00",
  "model": "ngvt_gaia_solver",
  "level": "all",
  "split": "validation",
  "total_questions": 450,
  "correct": 135,                    ← Count of correct answers
  "accuracy": 0.30,                  ← Accuracy percentage (0.30 = 30%)
  "by_level": {
    "1": {"correct": 100, "total": 150, "accuracy": 0.667},  ← Level 1
    "2": {"correct": 25, "total": 150, "accuracy": 0.167},   ← Level 2
    "3": {"correct": 10, "total": 150, "accuracy": 0.067}    ← Level 3
  },
  "performance": {
    "total_time_seconds": 3600.0,
    "avg_time_per_question_ms": 8000.0,
    "throughput_per_second": 0.125
  }
}

Key Metrics:
  accuracy: 0.0-1.0 (0.30 = 30%)
  correct: Number of correct answers
  total_time_seconds: Total evaluation time
  throughput_per_second: Questions evaluated per second
