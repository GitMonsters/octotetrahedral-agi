Accuracy
  1. Find 8-cell clusters — connected components using 8-connectivity. These are shape templates (diamonds, triangles, lines).
  2. Find indicator pairs — two adjacent cells with different non-zero colors, both isolated (no same-colored neighbors). The closer cell to the cluster = output color, the farther = source color.
  3. Compute entry point — one step from the output indicator toward the cluster center. This is the reference origin for template offsets.
  4. Build template — all 8-cell positions relative to the entry point.
  5. Stamp — at every source cell position, overlay the template in the output color. Uninvolved cells persist from input; 8s, indicators, and source cells are cleared.

Key Insights

  • 🔷 8-cells define geometric templates — diamonds stamp diamonds, triangles stamp triangles, lines stamp lines
  • 🎨 Indicator pairs encode color mapping — (source → output) with proximity determining roles
  • 📐 Entry point handles directionality — templates auto-orient based on indicator-to-cluster direction
  • 🔄 Multiple clusters = parallel operations — each cluster independently stamps its sources
  • 🧹 Uninvolved cells persist — only 8s, indicators, and sources are cleared
