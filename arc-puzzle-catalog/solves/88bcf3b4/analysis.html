📐 Discovered Rule: "Wall-Anchor Reflection"

Each grid contains one or more independent triplets of shapes:

  1. WALL — A straight line (horizontal or vertical) that stays fixed
  2. MOVER — A bent/staircase shape adjacent to the wall that gets repositioned
  3. ANCHOR — A smaller shape that stays fixed and defines the reflection profile

Transformation:

  1. The mover's junction cell (where it touches the wall) stays fixed
  2. The mover "flips" from extending away from the anchor to extending toward and past the anchor
  3. The perpendicular distance profile follows: dist = anchor_dist + 1 at anchor rows, with linear approach from junction and symmetric decrease beyond
  4. Output cell count ≤ input count (truncated at grid boundaries or when profile reaches zero)

🔬 Algorithm Details

Distance Profile Construction

For a vertical wall at column W with mover on the left side:

RegionDistance FormulaDescription
Junctiondist = 1Starting point, always 1 cell from wall
Approachdist = lerp(1, anchor_dist+1, t)Linear increase from junction to first anchor row
Anchordist = anchor_dist(row) + 1One step beyond anchor at each anchor row
Beyonddist = last_anchor_dist + 1 - kDecreases by 1 per row past last anchor

Multi-Triplet Handling

Test grids contain multiple independent (wall, mover, anchor) triplets. The solver:

  1. Splits each color into 8-connected spatial components
  2. Classifies: straight-line components → walls, bent components → potential movers
  3. Pairs each mover with its adjacent wall (4-connected adjacency check)
  4. Assigns remaining components as anchors to nearest wall-mover pair
  5. Applies the reflection transformation to each triplet independently
